<h1 align="center">Mounzil مُنزِل</h1>

### **About**
Mounzil is a download manager based on Mounzil which is a graphical user interface of [Aria2](https://github.com/aria2/aria2).


### **Features**
- Multi-segment downloading
- Scheduling downloads
- Download queuing
- and many more!
